# postcardsfrontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Docker

#### Build
```
docker build -t postcardsfrontend .
```
#### Run
```
docker run -it -p 8080:80 --rm --name postcardsfrontend postcardsfrontend
```
